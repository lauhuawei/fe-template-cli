# fe-template-cli
> 前端模板管理cli,支持模板选择、查看、新增、删除 


### fetpl 
默认读取模板库中的模板，选择对应模板插入本地

### fetpl list
查看模板库中的模板

### fetpl add [file path]
添加模板，默认当前路径文件，可指定文件路径

### fetpl del [file path]
删除模板，默认返回模板列表，选择删除的列表，可以指定删除模板
